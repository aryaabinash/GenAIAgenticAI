CODE EXAMPLE
================================================================================

Here's what the special character removal function does:

```python
import unicodedata

def remove_special_chars(text):
    # Step 1: Normalize Unicode (decompose special chars)
    text = unicodedata.normalize('NFKD', text)
    
    # Step 2: Keep only ASCII characters (a-z, 0-9, spaces)
    text = text.encode('ascii', 'ignore').decode('ascii')
    
    # Step 3: Clean up extra spaces
    text = ' '.join(text.split())
    
    return text
```
